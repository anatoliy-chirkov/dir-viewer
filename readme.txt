Программа для просмотра файлов в корневой директории сервера

Установка
1. В файле core/config.php прописать даные для базы;
2. В консоле набрать команду 'php core/setup.php' - создастся база и таблица;
3. Создать симлинк 'index.php' на '_/dirViewer/index.php' в директории куда смотрит сервер (ln -s .. ..) (если нет других index.php),
    Или подключить '_/dirViewer/index.php' в index.php файле в корне сервера (require ..),
    Или настроить сервер, чтобы он сомтрел на нужную папку.
4. Для удаления созданной базы выполнить: 'php /core/remove.php';
